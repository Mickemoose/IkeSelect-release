# IkeSelect Releases

This repository is the public download and auto-update channel for
**IkeSelect**, a drag-and-drop character-select-screen editor for
IKEMEN GO.

## Download

Get the latest build from the
[Releases](https://github.com/Mickemoose/IkeSelect-release/releases)
page:

- Windows: `IkeSelect.exe`
- Linux: `IkeSelect-Linux`
- macOS: `IkeSelect-macOS`

On Windows, place `IkeSelect.exe` inside your IKEMEN GO folder, next
to `Ikemen_GO.exe`, and run it. It picks up that installation
automatically.

## Updating

IkeSelect checks this repository for a newer release on startup, and
on demand from **Help > Check for Updates**. When a newer version is
available it offers to download and install it, then relaunches into
the new build.

## About IkeSelect

IkeSelect edits your `select.def` while preserving every comment,
blank line and extended parameter it does not change, so the roster
file is never silently rewritten. It includes a live select-screen
preview, drag-and-drop character placement, slot editing,
VSelect-style cell locking, a roster lineup view for comparing
sprite scale, automatic backups before every save, and built-in
help (press F1).
